# 财迹 FinTrack 使用手册

**版本 1.2 | 2025年6月 | 作者 lianghuaiyuan**

---

## 概述

财迹（FinTrack）是一款完全离线、支持多用户的个人财务管理应用。

- **无需联网** — 所有数据存储在本地 SQLite 数据库中
- **账户制管理** — 按资金账户记录余额/市值，无需逐笔记账
- **收支独立** — 收支记录与投资变动分开，准确计算 XIRR 年化收益率
- **定存自动化** — 单利按日计息，到期自动提醒
- **多用户** — 支持多账户共用，管理员可管理所有用户
- **局域网共享** — 启动后局域网内任意设备浏览器访问

## 快速开始

### 环境要求
- **Node.js >= 16**（[nodejs.org](https://nodejs.org) 下载 LTS 版本）
- Windows / macOS / Linux 均可

### 安装
1. 解压项目文件夹到任意位置
2. 双击 `setup.bat`（Windows）或运行 `npm install`（server 和 client 目录各一次）
3. 等待依赖安装完成

### 启动
```
双击 start.bat
```
启动后自动打开两个命令行窗口：
- **后端服务**（端口 3000）
- **前端服务**（端口 10004）

浏览器访问 **http://localhost:10004** 即可。

### 局域网访问
启动后，同一网络下的手机、平板、其他电脑均可访问：
```
http://<本机IP地址>:10004
```
例如 `http://192.168.1.100:10004`。本机 IP 可通过 `ipconfig`（Windows）或 `ifconfig`（Mac/Linux）查看。

## 页面功能说明

### 1. 仪表盘

登录后的首页，展示财务状况全景：

**统计卡片**：总资产、总收益、年化收益率(XIRR)、定期存款本息

**图表**（支持暗色/亮色主题）：
- **📈 资产变化趋势** — 30天/90天/1年可切换折线图，渐变填充，Y轴自动换算万元
- **🍩 资产分布** — 环形图，扇区外标签+引线，中心显示总额
- **📊 月度收支对比** — 柱状图，圆角柱顶，收入(绿)和支出(红)对比

**到期提醒**：7天内到期的定存显示黄色警告条。

### 2. 账户管理

创建、编辑、删除资金账户。每个账户包含：

| 字段 | 说明 |
|---|---|
| 账户名称 | 如"微信零钱""招商银行储蓄卡""中信证券" |
| 账户类型 | 零钱 / 银行 / 证券投资 / 其他 |
| 资产类别 | 零钱 / 定期存款 / 投资（决定在饼图中的分组） |
| 初始余额 | 仅创建时可设，之后通过资产列表调整 |

支持按类型筛选（全部 / 零钱 / 银行 / 证券投资 / 其他）。

### 3. 资产列表

以卡片形式展示所有账户余额，点击进入详情：

- **更新余额** — 直接覆盖为新值（如月末核对银行卡余额）
- **调整增/减** — 在当前余额上加减（如工资入账 +15000）
- **变动记录** — 时间线卡片展示历史变更，旧值 → 新值 + 变动额

### 4. 定期存款

创建银行定期存款，系统自动按单利每日计息：

- 本息和 = 本金 + 本金 × 年利率 × (已存天数 ÷ 365)
- 每个定存卡片显示：当前本息、已获利息、已存天数、到期倒计时
- 到期当天出现「赎回」按钮，赎回后本息自动转入关联账户
- 已赎回的定存在下方独立展示

### 5. 收支记录

独立于投资变动的日常收支记录：

- 收入分类：工资、奖金、兼职、投资、其他
- 支出分类：餐饮、房租、交通、购物、娱乐、医疗、教育、其他
- 支持按全部/收入/支出筛选
- 顶部汇总行：收入、支出、结余
- 月度统计表：最近12个月收支对比
- **导出 CSV**：可用 Excel 打开分析

### 6. 数据导入导出

- **导出全部**：下载 JSON 备份文件，包含所有账户、定存、收支、历史快照
- **导入备份**：选择 JSON 文件 → 预览内容 → 确认覆盖（⚠️ 不可撤销）
- 适合换电脑迁移或定期备份

### 7. 报表导出

三种导出格式，均通过浏览器下载（自动携带认证令牌）：

| 格式 | 内容 | 用途 |
|---|---|---|
| CSV | 所有收支明细 | Excel/Numbers 分析 |
| Excel | 账户余额 + 定存汇总 | 打印存档 |
| JSON | 全部数据完整备份 | 迁移到其他电脑 |

### 8. 账号管理

- 查看用户名和用户 ID
- **修改密码** — 需输入当前密码
- **删除账户** — 永久删除所有数据（需二次确认）
- **退出登录**

### 9. 使用说明

本文档的页面内版本，包含账户类型说明、快速上手步骤、收益率算法解释、备份迁移指南。

### 10. 用户管理（仅管理员）

管理员专有功能，普通用户不可见：

- **用户列表** — 查看所有注册用户的 ID、角色、注册时间、数据量统计
- **重置密码** — 为任意用户设置新密码（至少6位）
- **设为/取消管理员** — 赋予或撤销管理员权限
- **删除用户** — 删除用户及其所有数据
- 自己的行高亮显示，不可删除自己或取消自己的管理员权限

## 收益率计算（XIRR）

系统采用 XIRR（扩展内部收益率）算法，从以下数据构建现金流序列：

1. 投资类账户的资金流入/流出记录
2. 定期存款的本金投入和赎回
3. 当前总资产作为最终现金流入

用牛顿迭代法求解使 NPV=0 的年化收益率。收支记录不参与计算，结果反映纯投资收益。

---

## 定存利息计算

采用**单利按日计息**，与中国大陆银行定期存款计息方式一致：

```
本息和 = 本金 + 本金 × 年利率 × (已存天数 / 365)
```

## 数据备份与迁移

### 备份
进入「数据导入导出」→ 点击「导出全部」→ 下载 `.json` 文件保存到安全位置。

### 迁移到新电脑
1. 新电脑上安装并启动 FinTrack
2. 注册任意账号并登录
3. 进入「数据导入导出」→ 选择备份文件 → 预览确认 → 导入
4. 刷新页面即可看到所有数据

### 快速迁移（可选）
直接复制 `server/data/fintrack.db` 文件到新电脑的相同位置。

## 常见问题

**Q：启动报错"端口被占用"？**
A：关闭已运行的 FinTrack 进程，或在 `.env` 中修改 `PORT=3001`。

**Q：数据库出错？**
A：删除 `server/data/fintrack.db` 后重启，数据库会自动重建（⚠️ 数据会丢失）。

**Q：手机访问不了？**
A：确保手机和电脑在同一 Wi-Fi；检查防火墙是否放行端口 10004。

**Q：图表显示空白？**
A：先创建账户并添加一些余额变动记录，图表需要数据才能渲染。

**Q：如何重置管理员密码？**
A：删除 `server/data/fintrack.db` 后重启，admin 账号会被重新创建（密码 11235813）。
