💎 财迹 FinTrack v1.2

离线个人财务管理应用 · 作者 lianghuaiyuan · hrblianghuaiyuan@163.com

概述

财迹（FinTrack）是一款完全离线、支持多用户的个人财务管理应用。

  • 无需联网 — 所有数据存储在本地 SQLite 数据库中
  • 账户制管理 — 按资金账户记录余额/市值，无需逐笔记账
  • 收支独立 — 收支记录与投资变动分开，准确计算 XIRR 年化收益率
  • 定存自动化 — 单利按日计息，到期自动提醒
  • 多用户 + 管理员 — 支持多账户共用，管理员可管理所有用户
  • 局域网共享 — 启动后局域网内任意设备浏览器访问

快速开始

环境要求

Node.js >= 16（nodejs.org 下载 LTS 版本），Windows / macOS / Linux 均可。

安装与启动

  1. 双击 setup.bat 安装依赖（仅首次）
  2. 双击 start.bat 启动服务
  3. 浏览器访问 http://localhost:10004

其他设备通过局域网 IP 访问，如 http://192.168.1.xxx:10004。

📋 资产列表

  • 按账户展示余额，点击进入详情
  • 更新余额：直接覆盖为新值
  • 调整增/减：在现有值上加减（如工资入账 +5000）

设计说明

为什么用单利计息？

国内银行定期存款统一采用单利计算。单利按日计息透明直观，符合实际银行业务规则。

XIRR 收益率

系统构建现金流序列（投资账户调整记录 + 定存买入/赎回 + 当前总市值），使用牛顿迭代法求解使净现值为零的年化利率。收支记录不参与计算，结果反映纯粹的投资回报。

离线多用户

无邮箱/手机注册，仅需用户名+密码。bcrypt 加密存储不可逆。JWT 令牌 7 天有效。数据通过 user_id 外键完全隔离。

备份与迁移

  1. 旧电脑：导入导出 → 导出全部 → 下载 JSON
  2. 新电脑：安装启动 FinTrack → 创建任意账号
  3. 导入导出 → 选择备份文件 → 确认覆盖

也可直接复制 server/data/fintrack.db 文件。

常见问题

Q：端口被占用？
A：修改 .env 中的 PORT 或关闭占用进程。

Q：数据库错误？
A：删除 server/data/fintrack.db，重启自动重建。

Q：如何局域网访问？
A：Vite 已绑定 0.0.0.0:10004，其他设备用本机 IP + 端口号即可。

Q：普通用户看不到用户管理？
A：这是预期行为，只有管理员可见。用 admin 账号登录。
